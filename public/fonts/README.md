# Courier Prime Font

This folder contains the Courier Prime font family used throughout the application.

## Font Files

- `CourierPrime-Regular.ttf` - Regular weight
- `CourierPrime-Bold.ttf` - Bold weight
- `CourierPrime-Italic.ttf` - Italic style
- `CourierPrime-BoldItalic.ttf` - Bold Italic style

## Usage

The fonts are loaded via `@font-face` declarations in `src/index.css` and applied globally to the entire application.
